# ARG Design Spec: E-09 Protocol: Echo of the Forgotten (E-09 协议：遗忘之声)

## 1. 概述 (Overview)
这是一个深嵌在个人网站中的交互式 ARG（另类实境游戏），基于《裂月编年：破碎之光》世界观。游戏以旁支角色——真理财阀的实验体“迷失者”埃拉拉 (Elara, E-09) 为主线。ARG 包含跨页面解密、高度可探索的模拟终端界面、以及最终的道德抉择系统。

## 2. 叙事核心 (Narrative Arc)
- **幽灵主角**：埃拉拉 (Elara, E-09)，一个记忆正在被财阀抹除的迷失者。她的残存意识感染了网站代码。
- **反派机制**：守望者-7 (Watcher-7)，真理财阀的 AI 监控程序，试图诱导玩家清除异常数据。
- **核心秘密**：真理财阀为了实现“高维进化”，正秘密筹划人为引发“第13次大潮灾”，以此收割全大陆的生命与灵晶。

## 3. 游戏机制与流程 (Puzzle Mechanics & Flow)

### 阶段一：异象的萌芽 (The Disturbance)
- **触发位置**：诱饵博客文章《网页性能优化的几点思考》。
- **机制**：通过 CSS 禁用全站选中，唯独一句话（“我不是 Bug，我是被他们抹除的记忆”）允许被选中，且鼠标会变为文本输入（I-beam）光标。
- **线索**：当访问该页面时，F12 控制台输出红字警告：`[Veritas_AI_Watcher-7] DOM 树高维污染。实体 E-09 正在越权访问。强制清除通道：/quarantine-zone`

### 阶段二：终端探索与系统越权 (Terminal & Login)
- **位置**：隐藏路由 `/quarantine-zone`
- **机制**：一个全屏的、可自由探索的模拟命令行终端（真理财阀无菌风格）。
- **指令系统**：支持 `help`, `ls`, `cd`, `cat`, `clear`, `unlock` 等真实命令行操作，增强沉浸感。
- **第一层解密**：终端初始为锁定状态（Watcher-7 监控中）。埃拉拉的幽灵文字提示：“用我的真名覆盖系统”。玩家需在“关于我”页面的隐藏图片 `alt` 属性中找到 `[解剖对象 E-09：Elara]`。在终端输入 `ELARA` 后，获取文件系统访问权限。

### 阶段三：文件系统与三弦之锁 (The Three Echoes)
- **目录结构探索**：
  - `/sys_logs/`：包含几个补充世界观背景的普通日志（如对无菌肃清队的调度指令），玩家可使用 `cat` 阅读。
  - `/quarantine/`：包含最终目标文件 `E-09_memory_core.enc`。
- **解密触发**：使用 `cat E-09_memory_core.enc` 时，系统提示需进行三级解密，提示运行 `unlock E-09_memory_core.enc`。终端随之抛出三个指向“念、灵、渊”的谜语。

### 阶段四：跨页寻宝 (Scavenger Hunt)
玩家需在网站其他页面寻找三个“共鸣密钥”：
1. **【念】的密钥**：项目/作品页中有一串隐蔽的 ASCII 码 `112 104 121 115 105 99 97 108`（解码为 `physical`）。访问隐藏子页 `/physical`，获得密钥一：`FORM_SHATTERED`。
2. **【灵】的密钥**：连续快速点击网站的“明/暗模式切换键” 5 次，背景会闪现一张带有灵晶的瑟帕族 SVG 图腾，并弹窗提示密钥二：`BREATH_FROZEN`。
3. **【渊】的密钥**：访问任意错误路径进入 404 页面。页面全黑，鼠标化作“手电筒”（CSS 遮罩光照效果）。在黑暗的角落用光照亮埃拉拉的遗言，获得密钥三：`SOUL_LOST`。

### 阶段五：真相与抉择 (The Choice & Reward)
- **解锁真相**：在终端输入三把密钥，解开《黑曜石指令》档案，阅读财阀残酷阴谋的万字真相。
- **终局抉择**：Watcher-7 介入，强制弹出两个选项指令：
  - 输入 `purge`（归于寂静）：删除埃拉拉，掩盖真相。**奖励**：页脚永久添加低调的 `[Veritas Citizen / 真理良民]` 文本徽章。
  - 输入 `broadcast`（广播真相）：牺牲埃拉拉的最后意识，将阴谋大白于天下。**奖励**：终端碎裂，导航栏永久增加“碎月”图标，玩家可随时点击该图标开启或关闭整个网站的“白蚀（高维结晶）”赛博朋克主题。

## 4. 技术实施要点 (Technical Notes)
- **前端框架**：在现有 Vite + React 环境下，开发一个 Terminal 组件，维护独立的虚拟文件系统状态。
- **状态持久化**：终端的解锁进度、最终抉择的结果均保存在 `localStorage` 中。
- **CSS 特效**：404 手电筒效果使用 `mask-image: radial-gradient`。全局“白蚀”主题通过覆盖 CSS 原生变量（CSS Variables）并追加到 `html` 标签的数据属性中实现。
